# Задачи на условия

1. Напишите программу, которая проверяет, что для заданного четырехзначного числа выполняется следующее соотношение: сумма первой и последней цифр равна разности второй и третьей цифр.

Ввод:
```
1614
```
Вывод:
```
YES
```

Ввод:
```
1234
```
Вывод:
```
NO
```
